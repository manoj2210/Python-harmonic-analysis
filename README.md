# Python-harmonic-analysis
Harmonic Analysis of Fourier Series with PyQt5 
  
##Required Packages

* PyQt5
* Sympy
* Numpy
* Pyqtgraph
* Matplotlib

###Installation

```bash
python3 setup.py
``` 

###Usage
Make sure you have installed all the packages with setup.py

```bash
python internal/Ui_Start.py
```
